# 🛠️ Usage

Clone the repository:

```bash
git clone https://github.com/maheswari1829/DevOps-Scripts.git
```

Move into the project directory:

```bash
cd DevOps-Scripts
```

Run scripts:

```bash
chmod +x filename.sh
./filename.sh
```
